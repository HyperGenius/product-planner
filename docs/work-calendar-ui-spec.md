## 状態表示

### 通常の日付
- 背景色: 白
- テキスト色: 黒

### 休日として設定された日付
- 背景色: 薄い赤 (bg-red-200)
- テキスト色: 濃い赤 (text-red-900)
- フォント: 太字

### 臨時稼働日として設定された日付
- 背景色: 薄い青 (bg-blue-200)
- テキスト色: 濃い青 (text-blue-900)
- フォント: 太字

## 操作フロー

### 個別設定
1. カレンダー上の日付をクリック
2. ダイアログが開く
3. ステータス（稼働日/休日）を選択
4. 備考を入力（オプション）
5. 「保存」ボタンをクリック
6. データベースに保存され、カレンダーの表示が更新される
7. 成功メッセージ: "カレンダーを更新しました"

### 一括設定（土日を休日に設定）
1. 「土日を休日に設定」ボタンをクリック
2. 現在表示中の月の全ての土曜日・日曜日が自動的に休日として設定される
3. 成功メッセージ: "8件の日付を休日に設定しました"（件数は月による）

### 一括設定（土曜日を稼働日に設定）
1. 「土曜日を稼働日に設定」ボタンをクリック
2. 現在表示中の月の全ての土曜日が稼働日として設定される
3. 成功メッセージ: "4件の日付を稼働日に設定しました"（件数は月による）

## レスポンシブデザイン

- デスクトップ: カレンダーは中央に配置
- タブレット: カレンダーは画面幅に合わせて調整
- モバイル: カレンダーは縦に長く表示

## アクセシビリティ

- キーボード操作対応
- スクリーンリーダー対応
- カラーコントラスト WCAG AA 準拠
- フォーカス表示

## エラーハンドリング

### ネットワークエラー
- エラーメッセージ: "カレンダーの更新に失敗しました"
- トーストで表示
- エラーログをコンソールに出力

### 認証エラー
- 自動的にログイン画面にリダイレクト

### データ検証エラー
- 入力フィールドに赤枠を表示
- エラーメッセージをフィールド下に表示
